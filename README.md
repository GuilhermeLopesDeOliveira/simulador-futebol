# Simulador de Campeonato de Futebol

Simulação de 10 partidas em JavaScript com cálculo automático de pontuação (vitória = 3 pontos, empate = 1 ponto).

## Conceitos aplicados
- Estrutura de repetição (`for`)
- Condicionais (`if/else if`)
- Acumuladores

## Como executar
Abra o arquivo `futebol.html` no navegador e pressione F12 para abrir o console — os resultados aparecem lá.
